##Description
This application will serve as a cipher. It will recieve a word, the direction to encipher in and the number of letters to shift. It will output the word in a new drupal page.

##Technologies Used
Drupal
<br>
PHP
<br>
Github





##Use and Editing
To use the app, download the files from github. Open the index.html file in the browser of your choice. The webpage will open and allow the visitor to use the site.  <br>
